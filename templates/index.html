VULNSENSE

This is a web application for detecting vulnerabilities in smart contracts. The web application uses a tool which is presented in this paper.

The tool detects two types of vulnerabilities, Reentrancy and Arithmetic, in smart contracts written in the Solidity language and deployed on the Ethereum Blockchain.
